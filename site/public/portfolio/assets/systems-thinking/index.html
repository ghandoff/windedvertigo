systems thinking simulator
every system is perfectly designed to produce the results it gets.
ten institutions. ten mandates. the same five leverage tiers.

each scenario puts you on a committee with a mandate and a set of decisions. some change parameters. some change rules. some change who decides. one changes the question itself.

your job is not to find the right answer. it is to notice which kind of decision bends the system — and why.
systems dynamics leverage points stakeholder negotiation causal-loop reasoning paradigm recognition
how to play
  1. choose a scenario from the menu.
  2. assign roles — each person argues from their role's interest.
  3. read each intervention. expand the detail. discuss.
  4. reach consensus, then run it. watch the 25-year trajectory.
  5. after each run, read the "why" — it explains the leverage tier.
  6. after all five, look for the pattern across interventions.
