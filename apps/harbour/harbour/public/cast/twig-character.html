twig sketch — v3

  • Four poses: base, writer (marks in sand), fishing rod (over water with line + hook), walking stick (J-handle on ground).
  • Real path tween, not a cross-fade. Every pose defines a spine path and a fork path; both are sampled to points at load (getPointAtLength × 48 / 12), then lerped point-by-point with requestAnimationFrame over 900ms (easeInOutCubic). Same technique as Cord.
  • The static card symbols keep their full wood-grain detail; the morpher uses shared ink + bark + grain + blush paths so one d update redraws everything together.
  • During the tween: blur 3px, wobble filter is removed from outlines (the per-frame feDisplacementMap would jitter and cost too much), eyes + extras cleared. All restored on settle.
  • Decorations split into bg (water, ground line, cursive scribble — behind stick) and fg (knots, leaf bud, hook + line, dust, ripples — in front). Both appear on settle, never mid-morph.
  • Personality motion: .twitch (sharp 3-4° jerk every 7s), .tap (tiny impatient translate), .quickblink (4s cycle).
