how to pick, the short version

  • Pick A (Nunito only) if you trust illustration + colour + shape to carry the playfulness. Simplest, lowest risk, no new dependency. Fits the "crayon drawer, warm" vibe through warmth, not flourish.
  • Pick B (+ Fraunces) if you want the hero to feel hand-made and literary — the Oliver Jeffers tilt. Works because Fraunces soft/wonk isn't stunt-typography; it's calibrated warmth. Strongest pair with the ink + bark character rendering.
  • Pick C (+ Caprasimo) if you want maximum children's-book-cover energy. Strongest personality, loudest pair. Best if the rest of the UI is willing to sit back (quieter palette, fewer rotations, let Caprasimo be the thing).

My read: B is the best fit for the Sago Mini / Oliver Jeffers references and the character cast (the ink-and-bark rendering wants a hand-cut serif). A is the right default if you want to ship the refresh without adding a font dependency. C is fun but louder than your illustration layer needs help to be.
